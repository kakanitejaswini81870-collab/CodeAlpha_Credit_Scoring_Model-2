## 📋 Project Overview
This project predicts creditworthiness of loan applicants using Machine Learning algorithms. Banks use credit scoring to decide whether to approve loans or not.
## 🎯 Objective
Build a model that classifies applicants as Good Credit or Bad Credit based on financial details.
## 🛠️ Tech Stack
- **Python, Pandas, NumPy, Scikit-learn**
- **Models:** Logistic Regression & Random Forest
## 📊 Dataset
German Credit Dataset - 1000 applicants with 20 features
## 📈 Results
- **Logistic Regression:** ~75% Accuracy
- **Random Forest:** ~76% Accuracy
- **Key Features:** Credit Amount, Duration, Age
